Quizzical

A quiz app created using the MERN stack as part of the final project at Code Nation with my teammate Ethan.

Users can create an account, log in, select a category and level of difficulty for their quiz, then answer 10 questions retrieved using an API.

Scores and times can be submitted to a leaderboard which shows the current top ten ranking users.

Admin Page

An admin page which displays all users registered to a database, along with their usernames and email addresses.

The same site was made using both SQL and MongoDB databases.

City Forecast

A weather API app made using Node & Express.

Users can type in the name of a city and a country code to display the current weather in their chosen city. A list of example country codes is provided, which users can click to automatically insert into the search field.

Separate pages specifically for Manchester and London with seven day forecasts are also included.

Nutrition Top Trumps

A nutrition-themed game to played in the terminal, based on the popular card game.

A deck of thirty cards with different food items and their corresponding calories, carbs, fat and protein content is dealt between the player and computer.

Users must use their judgement to choose the category they think is most likely to trump the computer's card. The player (or computer) that wins all thirty cards wins.

Created to help practise using Javascript classes and array functions.
